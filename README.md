# 🏠 AI Interior Designer

An AI-powered web application that generates realistic interior design images based on user preferences.

---

## ✨ Features

* Generate interior designs using AI
* Select room type, style, and budget
* Custom prompt support
* Modern UI with live preview
* Fast image generation using Hugging Face models

---

## 🛠️ Tech Stack

* Python (Flask)
* HTML, CSS, JavaScript
* Hugging Face API (FLUX model)

---

## 🚀 How to Run

1. Install Python

2. Install required libraries:
   pip install flask requests python-dotenv

3. Run the app:
   python app.py

4. Open in browser:
   http://127.0.0.1:5000
